# Работа с mdadm

с добавлением дисков в оригинале возникли проблемы, пожтому был взят пример из   
<https://github.com/nixuser/zfs/blob/a661318f88dc44ea69b48bbfd52b8c3194f929a9/Vagrantfile>
(пример оттуда же с эксперементальной версией тоже в оригинале не отработал. По умолчанию в документации описано, что используется контроллер сата, по факту IDE с ограничением в 4 диска. Так что потребуется еще дополнительно создать сата контроллер.)

Добавлен скрипт для сборки рейда, создания раздела, партиций, монтирования с последующи добавлением в /etc/fstab

добавлен эхо lsblk в начале и в конце, а так же cat /etc/fstab и /etc/mdadm/mdadm.conf
для обновления системных файлов простое эхо не работает, поэтому использовался tee -a
